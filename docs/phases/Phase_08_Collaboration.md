# Phase 8 — Collaboration Modules

**Status:** **DEFERRED — not part of simple SVN plan**

---

## Why deferred

Simple SVN platform focuses on core VisualSVN operations:

- Users, groups, path permissions
- Checkout, Update, Commit, Diff, Log
- Web browse and Explorer menu

The following are **Gitea-like extras**, not standard SVN features:

- Issue tracking
- Kanban board
- Wiki
- SVN Review Request / Branch Merge Approval

Implement only if GMS explicitly requests collaboration features after MVP (Phase 6) is live.

---

## Original scope (for reference)

If revived later, this phase would cover issues, project board, wiki, and SVN Review Request workflow.

See [GMS_SVN_Platform_Detailed_Project_Document.docx](../../GMS_SVN_Platform_Detailed_Project_Document.docx) §8 for full specification.
